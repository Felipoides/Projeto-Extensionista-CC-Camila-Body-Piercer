# Camila Body Piercer - Site Institucional

Site institucional desenvolvido para a **Camila Body Piercer**, profissional de body piercing com 8 anos de experiencia atuando em **Varginha/MG**. O projeto foi criado como parte de uma disciplina extensionista do curso de Ciencia da Computacao, com o objetivo de oferecer presenca digital e visibilidade para o trabalho da profissional.

## Sobre o Projeto

O site apresenta os servicos, a trajetoria profissional e os canais de contato da Camila, permitindo que clientes conhecam seu trabalho, vejam o catalogo de piercings e entrem em contato facilmente via WhatsApp ou Instagram.

### Funcionalidades

- **Pagina Inicial (Home)** - Apresentacao da profissional com destaque para seus diferenciais: qualidade, seguranca/higiene e atendimento rapido.
- **Sobre** - Historia e trajetoria profissional da Camila no ramo de body piercing.
- **Servicos** - Catalogo completo dos tipos de perfuracao oferecidos (basica, avancada, intima e reconstrucao), com galeria de imagens dos trabalhos realizados.
- **Contato** - Formulario de contato e links diretos para WhatsApp e Instagram.
- **Acessibilidade** - Recursos de acessibilidade integrados (modo claro/escuro e outros ajustes visuais).

## Como Executar

Por se tratar de um site estatico, basta abrir o arquivo `index.html` em qualquer navegador. Nao e necessario instalar dependencias ou rodar um servidor.

```bash
# Clone o repositorio
git clone https://github.com/Felipoides/Projeto-Extensionista-CC-Camila-Body-Piercer.git

# Abra o arquivo principal no navegador
open index.html
```
