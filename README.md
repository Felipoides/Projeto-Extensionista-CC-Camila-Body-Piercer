# Projeto Extensionista CC - Camila Body Piercer

Projeto de site estático para o serviço de body piercing da Camila, desenvolvido com HTML,CSS e JavaScript.

## Descrição

Este projeto inclui páginas de apresentação, serviços, contato e sobre, além de um layout responsivo e estilizado para transmitir a identidade visual da marca.

## Estrutura do projeto

- `escritas HTML do projeto/`
  - `index.html` – página inicial
  - `Servicos.html` – página de serviços
  - `sobre.html` – página sobre a profissional
  - `contato.html` – página de contato
  - `style.css` – folhas de estilo do site
- `img/` – imagens utilizadas no projeto
- `README.md` – documentação do projeto

## Tecnologias

- HTML
- CSS
- Javascript
